Given a pointer to an ExpEvaluator on the host and a GPUExpEvaluator on the GPU, copy all of the properties from the ExpEvaluator object on the host to the GPU.

This routine is called by the GPUSolver::initializeExpEvaluator() private class method and is not intended to be called directly.
